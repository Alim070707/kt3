Задание «Адаптивный интерфейс»

Создайте новый проект Windows Forms в Visual Studio.
На главной форме создайте интерфейс с использованием следующих элементов управления:
- Панель (Panel), закрепленная слева с помощью свойства Dock.
- Рабочая область (Panel), закрепленная на оставшейся части окна с помощью свойства Dock.
Добавьте на рабочую область текстовое поле (TextBox) и кнопку (Button).
- Настройте свойства Anchor для этих элементов, чтобы они изменяли размер и оставались пропорционально размещенными при изменении размеров окна.
Запустите приложение и протестируйте адаптацию элементов управления при изменении размеров окна.
Продемонстрируйте результат. Для онлайна: прикрепите в виде скриншотов результаты проделанной работы.
